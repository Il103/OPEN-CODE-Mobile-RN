## ✨ Features

- 🌊 **Liquid Glass Interface** — Native glass morphism design with smooth animations
- 🤖 **Multi-Model AI** — GPT-4o, GPT-4, Claude 3.5 Sonnet, Claude 3 Opus, Gemini 2.0 Flash/Pro/Ultra
- 📷 **Image Support** — Send and analyze screenshots, code images, diagrams
- 🔐 **Google Sign-In** — Real Firebase authentication
- ⚡ **Real AI Integration** — Connect your own API keys for OpenAI, Anthropic, and Google
- 🎨 **3 Themes** — Dark, Light, Ocean with full Liquid Glass styling
- 📱 **Native Android** — Built with React Native, not a web wrapper

## 🔧 Setup

1. Download and install the APK
2. Sign in with Google or create an account
3. Add your API keys in Settings
4. Select your preferred model
5. Start coding

## 🏗️ Tech Stack

- **React Native** (Expo) — Native Android app
- **Firebase Auth** — Google Sign-In & email auth
- **Expo Image Picker** — Camera & gallery image support
- **React Native Reanimated** — Smooth animations
- **Zustand** — State management
